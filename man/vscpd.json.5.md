# NAME

vscpd.json - VSCP Daemon configuration file

# SYNOPSIS

/etc/vbscp/vscpd.json

# DESCRIPTION
The configuration file for the VSCPD daemon is a XML based file that set up the different functionalities of the VSCP daemon.
