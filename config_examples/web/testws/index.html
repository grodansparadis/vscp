If you want to learn more or need help

You find links to the different parts of the documentation for the VSCP & Friends project here. It's a lot of words because it's a big and advanced system with many features. You may also find our wiki to be a valuable resource. If you have questions or need pointers on where to start subscribe to our mailinglist. You will find that we have a very nice discussion climate and any question is taken care of.

Need VSCP hardware.


If you ned hardware you can build things yourself. The wiki contains many examples. You can also use any hardware you have have by building a simple driver for it. Check the documentation on how to do it or use any of the available drivers as a starting point.
